# Uusien tiedostojen lisääminen

tarkistetaan tämän hetken tila komennolla
```
git status
```

lisätään  yksi uusi tiedosto:
```
git add tiedoston-nimi
```
useamman tiedoston lisääminen:
```
git add tiedosto1 tiedosto2
git add tiedosto*
git add .
```

uuden kommitin tekeminen:
```
git commit -m "viesti"
